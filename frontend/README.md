# AI Healthcare Triage System - Frontend

This is the frontend part of the **AI Healthcare Triage System** capstone project.  
It is built using **React (Vite)** and provides a user-friendly interface for entering symptoms and interacting with the system.

---

## Tech Stack

- React (Vite)
- JavaScript (ES6+)
- HTML5
- CSS3
- Node.js & npm

## Setup Instructions

### 1. Clone repository
```bash
git clone https://github.com/Rutvi-Sheridan/AI-Healthcare-Triage-System.git
```

### 2. Go to frontend folder
```bash
cd AI-Healthcare-Triage-System/frontend
```

### 3. Install dependencies
```bash
npm install
```

### 4. Start development server
```bash
npm run dev
```

### 5. Open in browser
```
http://localhost:5173/
```
